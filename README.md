# project-worker (MCP server example)

This repository provides an example MCP (Model Context Protocol) server used as a Gemini CLI extension. It demonstrates a lightweight TypeScript implementation you can adapt to integrate with project management tools in the wild (Jira, Trello, Asana, GitHub Projects, Azure Boards, etc.).

## Quick start

Install dev dependencies and build:

```bash
npm install
npm run build
```

Run the tests or typecheck:

```bash
npm run typecheck
npm test
```

Lint and format:

```bash
npm run lint
npm run format
```

## Contributing

Open issues or PRs on GitHub. CI will run the build + typecheck + lint on every push.
